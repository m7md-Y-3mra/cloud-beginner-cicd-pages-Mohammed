## Description

This project is a beginner-friendly demonstration of implementing a Continuous Integration and Continuous Deployment (CI/CD) pipeline using GitHub Actions and GitHub Pages. It features a static HTML/CSS/JS website.

## How to Run Locally

1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/m7md-Y-3mra/cloud-beginner-cicd-pages-Mohammed.git
   ```
2. Navigate to the project directory:
   ```bash
   cd cloud-beginner-cicd-pages-Mohammed
   ```
3. Open `index.html` in any modern web browser (e.g., Chrome, Firefox, Edge) either by double-clicking it or right-clicking and selecting "Open with".
